# IceCare
IceCare est une plateforme innovante dédiée à la santé mentale. On vise à combiner technologie, science comportementale et soutien émotionnel pour offrir des outils simples, accessibles et sûrs pour tous. Notre vision est rendre la santé mentale accessible, grâce à une expérience simple, sécurisée et agréable.
